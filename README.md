Sistema de distribución desarrollado en C# y Visual Studio 2015, diseñado para gestionar productos, ventas y registros de tiendas de manera sencilla y eficiente. El programa incluye funciones de administración de datos, actualización de registros y manejo de inventario mediante una interfaz intuitiva orientada a proyectos académicos y pequeños negocios.

Características
Gestión de productos y stock
Registro y actualización de datos
Interfaz sencilla y fácil de usar
Integración con archivos para almacenamiento de información
Proyecto desarrollado en Windows Forms
English

Distribution system developed in C# using Visual Studio 2015, designed to manage products, sales, and store records in a simple and efficient way. The program includes data management features, record updating, and inventory handling through an intuitive interface focused on academic projects and small businesses.

Features
Product and stock management
Data registration and updating
Simple and user-friendly interface
File integration for information storage
Project developed with Windows Forms
